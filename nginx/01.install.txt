下载package安装
1.CentOS系统
1)添加nginx仓库源
[root@vultr ~]# vi /etc/yum.repos.d/nginx.repo
[nginx-stable]
name=nginx stable repo
baseurl=http://nginx.org/packages/centos/$releasever/$basearch/
gpgcheck=1
enabled=1
gpgkey=https://nginx.org/keys/nginx_signing.key
module_hotfixes=true

2)安装nginx
yum makecache
yum install nginx

3)启动nginx
systemctl start nginx
systemctl enable nginx

4)测试nginx
curl -I 127.0.0.1
HTTP/1.1 200 OK
Server: nginx/1.22.0


2.Ubuntu
1)安装nginx
sudo apt-get update
sudo apt-get install nginx

2)启动nginx
systemctl start nginx

3)测试nginx
curl -I 127.0.0.1



从source code进行安装
1.CentOS
1)安装依赖环境
yum install zlib-devel pcre-devel openssl-devel

2)下载source code
下载地址
curl -O https://nginx.org/download/nginx-1.22.0.tar.gz
** 下载列表参考: https://nginx.org/en/download.html

3)编译安装
tar -zxvf nginx-x.xx.x.tar.gz -C /tmp
cd /tmp/nginx-x.xx.x
./configure --with-http_ssl_module
make
sudo make install

4)启动nginx
ln -s /usr/local/nginx/sbin/nginx /usr/sbin/nginx
nginx

5)测试nginx
curl -I 127.0.0.1
HTTP/1.1 200 OK
Server: nginx/1.22.0
