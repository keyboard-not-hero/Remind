下载package安装
1.CentOS系统
1)添加nginx仓库源
[nginx-stable]
name=nginx stable repo
baseurl=http://nginx.org/packages/centos/$releasever/$basearch/
gpgcheck=1
enabled=1
gpgkey=https://nginx.org/keys/nginx_signing.key
module_hotfixes=true

2)安装nginx
yum install nginx

3)启动nginx
nginx

4)测试nginx
curl -I 127.0.0.1
HTTP/1.1 200 OK
Server: nginx/1.22.0


从source code进行安装
1)下载source code
下载地址: https://nginx.org/en/download.html
例. 下载nginx-1.22.0.tar.gz
curl -O https://nginx.org/download/nginx-1.22.0.tar.gz

2)解压缩
tar -zxvf nginx-x.xx.x.tar.gz -C /tmp

3)安装必要软件
yum install zlib-devel pcre-devel
curl -O https://www.openssl.org/source/openssl-3.0.5.tar.gz
tar -zxvf openssl-3.0.5.tar.gz -C /usr/local

4)编译安装
cd /tmp/nginx-x.xx.x
./configure --with-http_ssl_module --with-openssl=/usr/local/openssl-3.0.5
make
sudo make install

5)启动nginx
ln -s /usr/local/nginx/sbin/nginx /usr/sbin/nginx
nginx

6)测试nginx
curl -I 127.0.0.1
HTTP/1.1 200 OK
Server: nginx/1.22.0
