ubuntu 22.04
1.安装必须条件
virtualbox开发环境
gcc
make
kernel header file

virtual GUI:
qt >= 5.3.2 或 qt>=5.6.2
sdl >= 1.2.7

虚拟机系统与外部硬件交互
vboxdrv module

虚拟机网络
vboxnetflt module
vboxnetadp module
** vboxdrv/vboxnetflt/vboxnetadp包含在virtualbox-dkms中

sudo apt-get install gcc make 
sudo apt-get install libqt5core5a libsdl1.2-dev 


2.安装virtualbox
下载地址: https://www.virtualbox.org/wiki/Linux_Downloads
sudo dpkg -i virtualbox-xxx


3.将用户加入vboxusers群组
sudo usermod -a -G vboxusers <username>
