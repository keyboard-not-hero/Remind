一、网站搭建
LAMP(Linux+Apache+MariaDB+PHP)是基于Linux服务器动态网站的基本套件，部署wordpress之前我们需要先安装好套件环境。

1.安装Apache:

[root@vultr ~]# apt-get install apache2

[root@vultr ~]# systemctl start apache2

[root@vultr ~]# systemctl enable apache2

**浏览器输入127.0.0.1(远程主机用服务器ip)测试是否出现test123欢迎界面，出现则成功安装


2.安装MariaDB:

[root@vultr ~]# apt-get install mariadb-server mariadb-client

[root@vultr ~]# systemctl start mariadb

[root@vultr ~]# systemctl enable mariadb

[root@vultr ~]# mysql_secure_installation


3.安装php:

[root@vultr ~]# apt-get install php php-mysql

# 将7.2替换为php版本号
[root@vultr ~]# vi /etc/php/7.2/apache2/php.ini
cgi.fix_pathinfo=0

[root@vultr ~]# systemctl restart apache2

[root@vultr ~]# vi /var/www/html/info.php
<?php
  phpinfo();
?>
**浏览器栏输入remote_ip/info.php，测试是否出现php欢迎界面


wordpress环境部署：

1.官网下载wordpress，并解压缩:
[root@vultr ~]# wget https://wordpress.org/latest.tar.gz

# 解压缩wordpress
[root@vultr ~]# tar -zxvf latest.tar.gz -C /var/www/html

#将wordpress文件夹内容移动到/var/www/html下
[root@vultr ~]# cd /var/www/html/wordpress && mv * .. && mv .htaccess ..

2.修改/var/www/html目录所属(递归):
[root@vultr ~]# chown -R www-data:www-data /var/www/html

3.创建数据库和账号:

[root@vultr ~]# mysql -u root -p

# 创建数据库
>create database wordpress;

# 添加相关帐号并授权(语法：GRANT ALL PRIVILEGES ON database.table TO user@host IDENTIDIED BY passwd)
>GRANT ALL PRIVILEGES ON wordpress.* TO user@localhost IDENTIFIED BY passwd;

4.wordpress站点设置(与数据库连接)：
浏览器输入ip/wordpress/wp-admin/install.php，step by step进行设置

三、HTTP加密
部署HTTPS

1.添加PPA
[root@vultr ~]# apt-get update

[root@vultr ~]# apt-get install software-properties-common

[root@vultr ~]# add-apt-repository universe

[root@vultr ~]# add-apt-repository ppa:certbot/certbot

[root@vultr ~]# apt-get update

2.下载certbot
[root@vultr ~]# apt-get install certbot python-certbot-apache

3.安装证书
[root@vultr ~]# certbot --apache

4.更新证书
[root@vultr ~]# certbot renew

5.验证证书是否工作
进入地址：https://www.ssllabs.com/ssltest/，输入域名
