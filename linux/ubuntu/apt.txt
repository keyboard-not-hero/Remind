apt - 包管理器的指令行接口


语法
apt [option] <action>

apt {-v | -h}


参数释义



action列表
autoremove
	移除根据依赖安装, 但是当前没有被依赖的软件
	可使用apt-mark manual/auto <pkg>指定软件为手动/自动安装

edit-sources
	使用预设的编辑器编辑/etc/apt/sources.list文件

full-upgrade
	类似于upgrade
	需要卸载其他软件进行更新时，执行该操作

install <package> [{=<pkg_version> | /<release_name>}] ...
	安装/更新指定软件
	package可以为str或通配符模式
	release_name可以为释出代号(buster/bullseye...)或分支(stable/testing...)

list
	所有可获得的软件列表
	配合--installed选项, 表示已安装的软件列表
	配合--upgradeable选项, 表示可更新的软件列表
	配合--all-versions选项, 表示所有可获得的软件列表

purge <package>
	移除指定软件, 并且移除配置文件

reinstall <package>
	重新安装指定软件

remove <package>
	移除指定软件, 但保留配置文件

satisfy
	

search

show

update
	更新源信息

upgrade
	安装所有可获得更新的软件(从/etc/apt/sources.list)
	该操作自动安装依赖的软件
	该操作不会移除已安装的软件
	当需要卸载其他软件进行更新时，不会执行该软件更新
