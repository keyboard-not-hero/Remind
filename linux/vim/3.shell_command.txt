指令：
vim

格式：
vim [options] <file_name1> [file_name2] ...

参数释义：
--version    列出版本信息及支持的特性

-A    以Arabic模式启动(文字方向为从右向左)

-e    以Ex模式启动(vim批处理)

-g    打开GUI版vim

-M    以只读方式开启文件(不可修改文件)

-n    不生成swap文件

-o    以水平分割窗口开启多文件

-O    以竖直分割窗口开启多文件

-R    以只读方式开启文件(可修改文件，并使用:w!强制保存修改)

-r    从交换文件恢复未保存修改(swap file)

-s    silent模式(只适用于-e模式)

-S <session_file>    指定以session文件恢复session信息

-x    加密文本
