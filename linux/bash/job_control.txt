jobs - 查看当前shell的job列表

语法
jobs [-lnprs] [<job_spec>]
jobs -x command [arguments]

参数列表
-l
    额外显示process ID

-n
    只显示上次输出状态后, 更改状态的job

-p
    只显示process ID

-r
    只显示Running状态的job

-s
    只显示Stopped状态的job
