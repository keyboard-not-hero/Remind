diff - 逐行比对文件差异

语法
diff [options] files

参数解析：
--normal  
    打印普通差异，默认比较格式

-q, --brief
	文件之间有差异进行简短报告

-s, --report-identical-files
	当文件完全一样时进行报告

-c, -C <num>, --context[=num]
	打印差异内容的上下文行数，默认为3

-u  
	使用统一上下文进行比较，常用

-e  
	ed脚本格式逐行比较

-y  
	将两个文件的完整内容分为两列进行比较

-r, --recursive
    对递归子目录下的文件进行比较


diff -u结果格式解析：
---  后接原始文件元数据
+++  后接目标文件元数据

-  后接只有原始文件包含的行
+  后接只有目标文件包含的行
[space]  后接原始文件和目标文件都包含的行

@@ -start1,number1 +start2,number2@@
@@  代表单次比较限定行范围
start1  代表原始文件本次比较的起始行
number1  代表原始文件本次比较范围的总行数

** diff默认可以比较两个目录下的文件内容和子目录名称，但不递归到子目录下的文件
