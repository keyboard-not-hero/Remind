more - 分页文件内容查看器

语法
more [options] file

参数释义
-<number>
	指定一页显示的行数

-d
	在左下角添加提示文字

-l
	more一般在换页符(^L)后暂停
	此选项阻止该特性

-f
	一页输出指定的逻辑行(占据多行的无换行符号内容为一行)
	默认为占据一页的显示效果行

-s
	将多个连续的空行替换为一个空行

-u
	不显示下划线

+/<str>
	在more中搜寻指定字符串

+<num>
	从指定行开始显示内容


env command:
h/?
	command相关帮助信息

<space>
	翻页

<num>z
	显示下一个num行
	默认为当前页面行数
	num指定后, 会成为后续默认值

<num><return>
	显示下一个num行
	默认为1
	num一次性有效

q/Q
	退出more环境

f
	跳过一页内容

b
	往回翻一页

!command or :!command
	在子shell中执行指令

:f
	当前文件名和所在行

.
	重复上一个指令
