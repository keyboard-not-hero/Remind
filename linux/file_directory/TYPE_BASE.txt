文件类型可以由ls -al的第一个字元得出, 类型如下:
1.-(regular file)
	存在内容的文件. 可分为以下类型:
	1)ASCII file - 使用ASCII编码的纯文本内容. 可使用cat直接查看内容
	2)binary file - 可执行程序. 可使用od执行查看
	3)data file - 给特定程序提供信息的文件. 如: /var/log/wtmp使用last程序, /var/lib/mlocate/mlocate.db使用locate程序

2.d(目录)
	包含其他文件或子目录的目录
	
3.l(链接文档)
	指向其他文件的软链接

4.b(区块设备)
	用于存放资料的设别, 通常位于/dev目录下. 如: 硬盘

5.c(字元设备)
	用于一次性读取的输入设备, 通常位于/dev目录下. 如: 鼠标/键盘

6.s(套接口文件)
	网络连接接口, 通常位于/run目录下

7.p(管道文件)
	多个程序读取同一个文件的解决方案


** 也可使用file指令查看文件类型
