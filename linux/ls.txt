ls - 目录/文件信息列表

语法：
ls [options] [file]...

参数释义：
-a, --all    列出的文档包含隐藏文档(以.开头的文件)，不排除.和..目录

-A, --almost-all    列出的文档包含隐藏文档(以.开头的文件)，排除.和..目录

--author    列出文件的创建者，需配合-l使用

-b, --escape    使用C-style跳脱符表示非打印字符(如空格为'\ ')，默认使用''限定

--block-size=<size>    文件大小单位(如:"K""M""G")，配合-l使用

-B, --ignore-backups    忽略备份文件(以~结尾)

-c    配合-lt使用，显示ctime并根据ctime排序；
      配合-l使用，显示ctime并根据name排序；
      单独使用，根据ctime进行

-C    以列的顺序显示列表，可消除-l的作用

--color[=args]    输出进行颜色区分，args可选：always(默认)/auto/never

-d, --directory    显示目录本身信息，而非目录包含的内容信息

-D, --dired    信息用于Emacs dired模式

-f    结果不排序，隐含-aU选项，消除已配置-ls --color选项

-F, --classify    根据文件类型添加后缀(*/=>@|)。例：目录后缀/，可执行文件后缀*

--file-type    根据文件类型添加后缀(/=>@|)

--format=<word>    以单词<word>来定义输出格式
  映射关系：
  across  -x
  commas -m
  horizontal -x
  long -l
  single-column  -1
  verbose -l
  vertical -C

--full-time    打印详细日期和时间，类似于-l --time-style=<full-iso>

-g    类似于-l，但不打印文件所有者字段

--group-directories-first    显示内容，目录先于文件

-G, --no-group    当结合-l时，不打印group

-h, --human-readable    当结合-l时，目录(或文件)大小单位根据文件大小调整(进制为1024)

--si    类似于-h，但进制为1000

-H, --dereference-command-line    当option后的file为symbolic链接时，显示被链接文件的信息，需配合-l使用

--dereference-command-line-symlink-to-dir    类似于-H，但用于目录引用

--hide=<pattern>    不显示被pattern匹配的项(pattern为shell通配符模式)，该选项被-a和-A覆盖

--indicator-style=<word>      根据类型文件添加后缀，使用<word>指定对应参数
  参数映射
  none
  slash  -p
  file-type  --file-type
  classify  -F

-i, --inode    显示inode号码

-I, --ignore=<pattern>    不显示被pattern匹配的项(pattern为shell通配符)

-k, --kibibytes    使用1k byte作为单位

-l    显示entry详细信息

-L, --dereference    当entry为symbolic link时，显示被链接的实际文件信息    

-m    单行显示多个entry，entry之间使用','分隔。-l选项被覆盖    

-n, --numeric-uid-gid    user和group使用ID形式显示

-N, --literal    不使用限定符('')打印含空格的entry    

-o    类似于-l，但不打印group

-p, --indicator-style=slash    使用'/'后缀标记目录

-q, --hide-control-chars    使用'?'代替非打印字符

--show-control-chars    显示非打印字符

-Q, --quote-name    使用""限定entry

--quoting-style=<word>    使用名称<word>指定限定字符 
  映射关系：
  literal  -N
  locale  -Q
  shell  none
  shell-always  '' 
  shell-escape  none
  shell-escape-always  ''
  c  -Q
  escape  -b

-r, --reverse    以倒序显示结果

-R    递归罗列子目录

-s, --size    根据block为大小单位显示文件大小。无-l选项时，新增列

-S    根据文件大小排序，最大优先

--sort=<word>    使用单词指定排序规则
  映射关系：
  none  -U
  size  -S
  time  -t
  version  -v
  extension  -X

--time=[atime|ctime]    显示modification/access/status time，默认为modification time(mtime)

--time-style=<style>    指定显示时间格式，可选：full-iso,long-iso,iso,locale,+<format>(该选项类似于date指定时间格式)

-t    根据mtime排序，最新优先

-T, --tabsize=<cols>    指定制表符<Tab>所占的列


-u    配合-lt使用，显示atime并根据atime排序；
      配合-l使用，显示ctime并根据name排序；
      单独使用，按ctime进行排序，最新优先

-U    entry不进行排序

-v    按版本进行排序，通常用于同名不同版本([~.0-9A-Za-z]参与比较，优先级：~>.>连续数字>字母)

-w, --width=<cols>    限制每一行可打印的字符数

-x    以行排序代替默认的列排序

-X    根据拓展名进行排序

-Z, --context    显示安全上下文(SELinux)

-1    每个entry占据一行

--help    帮助信息

--version    版本信息

**三种时间比较：
status time(ctime):文件权限/所有者/群组/内容改变
modification time(mtime):文件内容被改变
access time(atime):文件被访问
