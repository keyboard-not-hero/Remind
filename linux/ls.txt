ls - 目录/文件信息列表


语法：
ls [options] [file]...


options参数
-a, --all    
    列出所有的文档
    包含隐藏文档(以.开头的文件)

-A, --almost-all    
    列出所有的文档
    包含隐藏文档(以.开头的文件), 但不包含.和..

--author
    列出文件的创建者, 需配合-l使用

-b, --escape
    使用C语言格式跳脱符, 表示非打印字符(如空格为'\ ')

--block-size=<size>
    指定文件大小显示单位, 配合-l使用
    列表如下:
        K/M/G/T/P - 1024进制
	KB/MB/GB/TB/PB - 1000进制

-B, --ignore-backups
    不显示备份文件(以~结尾)

-c    
    配合-lt使用, 显示ctime并根据ctime排序
    配合-l使用, 显示ctime并根据name排序
    其他情况, 根据ctime进行排序

-C    
    只显示文件名. 默认选项

--color[=args]
    以不同颜色输出不同类型文件
    args可选列表:
        always - 一直使用颜色进行区分
	auto - 自动判别是否使用颜色区分
	never - 不使用颜色进行区分

-d, --directory    
    显示目录本身信息, 而非目录内的文件信息

-D, --dired    
    生成的信息用于Emacs dired模式

-f    
    多个参数结合模式. 包含的参数如下:
        -a - 显示隐藏文件
	-U - 内容不排序
    不包含的参数如下:
        -l - 显示详细信息
	-s - 显示文件大小
	--color - 以颜色区分文件类型

-F, --classify    
    根据文件类型添加后缀(*/=>@|)

--file-type    
    根据文件类型添加后缀(/=>@|)

--format=<word>    
    以单词<word>来定义输出格式, 映射关系如下:
  	across/horizontal  -x
	vertical -C
	    一列文件名(或目录名)代表一项，使用空格分隔
	commas -m
	    一列文件名(或目录名)代表一项，使用','分隔
	single-column  -1
	    一行文件名(或目录名)代表一项
	long/verbose -l
	    一行文件名(或目录名)的详细信息代表一项

--full-time
    YYYY-MM-DD hh:mm:ss.<ms> <timezone>时间格式
    类似于-l --time-style=full-iso

-g
    类似于-l, 但不显示文件所有者字段

--group-directories-first
    先显示目录, 再显示文件
    默认先文件，后目录

-G, --no-group
    当结合-l时, 不显示文件所属组字段

-h, --human-readable    
    以易读方式调整大小单位，需配合-l或-s使用
    目录(或文件)大小单位根据文件大小自动调整(进制为1024)

--si    
    类似于-h, 使用进制1000

-H, --dereference-command-line    
    当明确指定的file为symbolic链接时, 显示实际文件的信息

--dereference-command-line-symlink-to-dir
    类似于-H，只有软链接为目录时有效

--hide=<pattern>
    不显示被pattern匹配的项(pattern为shell通配符模式)
    该选项被-a和-A覆盖

--hyperlink[=<args>]
    是否对文件名进行超链接. 列表如下:
        always - 指定进行超链接
	auto - 自动判定是否进行超链接
	never - 不进行超链接

--indicator-style=<word>      
    根据类型文件添加后缀, word列表:
        none - 无后缀. 默认选项
	slash - 添加\，类似于-p选项
	file-type - 添加文件类型符号，类似于--file-type选项
	classify - 添加分类符号，类似于-F选项

-i, --inode
    显示inode索引号码

-I, --ignore=<pattern>
    不显示被pattern匹配的项(pattern为shell通配符)
    与--hide的差别在于, 当前参数不被-a与-A覆盖

-k, --kibibytes    
    使用1k byte作为单位

-l    
    显示文件和目录的详细信息
    一行代表一个文件(或目录)项

-L, --dereference
    当目标为软链接时, 显示实际文件信息

-m
    当每行显示多个项时，使用','分隔

-n, --numeric-uid-gid
    类似于-l, 但所有者和群组使用ID形式显示

-N, --literal    
    文件名按元格式显示
    文件名包含不可打印字符时，不使用默认的''进行限定

-o
    类似于-l, 但不显示群组

-p, --indicator-style=slash
    使用'/'后缀标记目录

-q, --hide-control-chars
    使用'?'代替非打印字符

--show-control-chars
    显示非打印字符

-Q, --quote-name
    将文件名放置在""内

--quoting-style=<word>
    使用名称<word>指定限定字符, word列表如下:
	literal - 元格式，不使用限定符，类似于-N
	locale - 按本地编码格式. 如: 中文环境使用全角单引号限定
	shell - shell默认格式，只有包含非打印字符的文件名使用''限定
	shell-always - 使用shell默认格式，但所有文件名都使用''限定 
	c - 使用""限定文件名，类似于-Q
	escape - 使用C语言格式跳脱符，类似于-b

-r, --reverse
    以原排序方式的倒序排列

-R
    递归显示子目录文件(或目录)，直到没有子目录(穷尽)

-s, --size    
    根据block大小为单位，显示文件大小

-S    
    根据文件大小排序, 最大优先

--sort=<word>    
    按指定规则排序, word列表如下:
	none - 不排序，类似于-U
	size - 根据文件大小排序，类似于-S
	time - 根据-l显示的时间进行排序，类似于-t
	version - 根据文件名中的版本格式数字进行排序，类似于-v
	extension - 根据文件名的拓展名进行排序，类似于-X

--time=[atime|ctime]    
    显示指定类型时间. 类型如下:
        atime - 最后一次访问时间(access time)
	ctime - 最后一次文件元数据修改时间(文件属性，包含文件内容被修改，change time)
	mtime - 最后一次文件内容修改时间(modification time)，默认显示类型，不可指定

--time-style=<style>    
    指定显示时间格式, 可选列表:
	full-iso - YYYY-MM-DD hh:mm:ss.<ms> <timezone>格式
	long-iso - YYYY-MM-DD hh:mm格式
	iso - MM-DD hh:mm格式
	locale - 根据本地语言环境输出时间格式
	+<format> - 自定义时间格式. 参考man 1 date

-t    
    根据时间进行排序, 最新优先

-T, --tabsize=<cols>    
    指定制表符<Tab>所占的列
    默认为8

-u    
    配合-lt使用, 显示atime并根据atime排序；
    配合-l使用, 显示atime并根据name排序；
    其他情况, 按atime进行排序

-U
    不对文件名进行排序

-v    
    按文件名中版本格式数组进行排序
    [~.0-9A-Za-z]参与比较, 优先级：~ > . > 连续数字 > 字母)

-w, --width=<cols>    
    每一行的字符数

-x    
    以行排序代替默认的列排序

-X    
    根据文件拓展名进行排序

-1    
    每个文件名占据一行

--help    
    帮助信息

--version    
    版本信息



SELinux选项:
--lcontext
	显示安全上下文
	隐式包含-l参数

-Z, --context    
	显示安全上下文

--scontext
	只显示安全上下文和文件名称
