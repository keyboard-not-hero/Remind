docker container - 管理container

分指令集合
docker container attach [option] <container>
	将已运行的container与stdin/stdout/stderr连接

docker container cp [option] <container>:<src_path> <host_dst_path>
docker container cp [option] <host_src_path> <container>:<dst_path>
	在container和主机之间拷贝文件
	container可以是running/stopped状态
	
docker container create [option] <image_name> [command] [args]
	创建一个container，但运行

docker container inspect [option] <container>...
	显示单个或多个container的详细信息

docker container kill [option] <container>...
	杀死一个或多个container

docker container ls [option]
	罗列container. 类似于docker ps
	默认只罗列running container
