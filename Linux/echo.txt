echo--显示一行文本

语法：
echo [OPTION] [STRING]

参数释义：
-n    不打印行尾换行符

-e    反斜杠(\)作为特殊字符

-E    反斜杠作为一般字符(默认)

--help    帮助文本

--version    打印版本信息

当-e生效时，以下特殊字符含义：
\\     backslash

\a     alert (BEL)

\b     backspace

\c     produce no further output

\e     escape

\f     form feed

\n     new line

\r     carriage return

\t     horizontal tab

\v     vertical tab

\0NNN  byte with octal value NNN (1 to 3 digits)

\xHH   byte with hexadecimal value HH (1 to 2 digits)
