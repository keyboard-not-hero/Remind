apt--command-line interface

参数释义：
update    更新容器的可安装包信息(从/etc/apt/sources.list)

upgrade    更新已安装软件，不会卸载软件，当需要卸载其他以更新时，不会执行该软件更新

install    安装软件

remove    移除软件，但保留配置文件

purge    移除软件，并删除配置文件

autoremove    移除软件，此类软件以前根据依赖安装，并且现在不被其他软件所依赖

list    查看可获取的软件列表名称

show    查看软件的详细信息

search    查找匹配关键字的容器内软件，并附带间接

source    下载源码(需配置原配置文件的deb-src使用)


apt-get--APT package handling utility

参数释义：
update    更新容器的可安装包信息(从/etc/apt/sources.list)

upgrade    更新已安装软件，不会卸载软件，当需要卸载其他以更新时，不会执行该软件更新

dist-upgrade    更新可更新软件，当需要卸载其他以更新时，进行卸载后更新

check    探测依赖是否完整

download    下载二进制文件到当前目录

clean    删除容器的缓存安装包信息

autoclean    清除无效的缓存安装包信息(保留有效信息)

-f install   修复问题

软件包：software-properties-common
add-apt-repository--Adds a repository into the /etc/apt/sources.list or /etc/apt/sources.list.d or removes an existing one
参数：
    添加指定源到/etc/apt/sources.list(优先)或/etc/apt/sources.list.d(ppa:<user>/<ppa-name>)目录下
-r    从/etc/apt/sources.list.d目录下删除指定源
-y    确认所有选项
-u    添加源后，更新源的可用软件信息(cache)，避免全局apt-get update



配置路径：/etc/apt/sources.list或/etc/apt/sources.list.d/
