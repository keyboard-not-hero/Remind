一、图形模式(部分分区可能无法删除)
1.右键单击'此电脑' -- 选择'管理'

2.双击'存储' -- 单击'磁盘管理'

3.找到对应磁盘的对应分区, 右键点击, 选择'删除卷轴'



二、指令模式(可强制删除所有分区)
1.使用管理员身份打开cmd

2.进入diskpart模式
指令: diskpart

3.显示磁盘列表
指令: list disk

4.选择磁盘号
指令: select disk <n>

5.显示分区列表
指令: list partition

6.选择分区
指令: select partition <n>

7.删除指定分区
指令: delete partition

** 如果删除多个分区, 重复步骤6/7
