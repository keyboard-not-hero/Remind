Demo_01 -- 字体类型
\documentclass[UTF8,fontset=ubuntu]{ctexart}
\setlength{\parindent}{0mm}
\begin{document}
\textnormal{范例文本, Demo text, 常规体, normal text}\\
\textrm{范例文本, Demo text, 罗马体, roman family}\\
\textmd{范例文本, Demo text, 中序体, medium series}\\
\textup{范例文本, Demo text, 直立体, upright shape}\\
\textsf{范例文本, Demo text, 等线体, sans serif family}\\
\texttt{范例文本, Demo text, 等宽体, typewriter family}\\
\textit{范例文本, Demo text, 斜体, italic shape}\\
\textsl{范例文本, Demo text, 倾斜, slanted shape}\\
\textbf{范例文本, Demo text, 加粗体, boldface seried}\\
\textsc{范例文本, Demo text, 小型大写, small caps shape}\\
\end{document}

内容讲解
1.\textrm{}为罗马字体
\textnormal、\textmd、\textup效果与\textrm一致

2.\textsf{}为等线体, 等线体字体线条一致, 没有衬线

3.\texttt{}为等宽体, 在中文(或英文)中, 所有字体(或字符)的宽度一致, 适用于打印

4.\textit{}为斜体

5.\textsl{}为倾斜体, 字体倾斜, 类似于\textit, 但字体更为方正

6.\textbf{}为加粗体

7.\textsc{}为字体更小的大写字母

8.\text开头的普通文本指令可位于任何环境中, paragraph/LR/math mode



Demo_02 -- 音节重音
\documentclass{article}
\setlength{\parindent}{0mm}
\begin{document}
\={o}u\\
\'{o}u\\
\v{o}u\\
\'{o}u\\

\b{i}u\\
\b{\i}u\\
\end{document}

内容讲解
1.\'{<letter>}用于在字符上进行注音

2.i/j由于头顶本来有'.', 所以需要去除, 格式为\i、\j

3.其他注音符号参考symbol_summary的表3. 适用于paragraph mode和LR mode

Demo_04 -- 下划线
\documentclass{article}
\usepackage{ulem}
\setlength{\parindent}{0mm}
\renewcommand{\ULthickness}{1pt}
\setlength{\ULdepth}{10pt}
\begin{document}
This is line \underline{one}\\
This is line two\\
This is line \underline{\phantom{three}}\\
This is line \uline{four}
\end{document}

内容讲解
1.\underline{<content>}用于给指定文本增加下划线, 但内容进入左右模式, 不能换行. 适用于文本模式和数学模式

2.\uline{<content>}用于给指定文本增加下划线, 并且内容可以换行, 可以修改下划线粗细和下划线距离文本的距离, 包含在宏包ulem中. 相关指令
    \ULthickness - 用于设置下划线的粗细, 默认为0.4pt, 配置命令: \renewcommand{\ULthickness}{<length>}
    \ULdepth - 设置下划线与文本的距离, 与文本尺寸相关, 配置命令: \setlength{\ULdepth}{<length>}

Demo_06 -- 破折号
\documentclass{article}
\begin{document}
    An intra-word dash or hyphen, as in X-ray.\\

    A medium dash for number ranges, like 1--2.\\

    A punctuation dash---like this.
\end{document}

内容讲解
1.-为最短的破折号，常用于连接字母

2.--为稍宽的破折号，用于表示数字范围

3.---为最宽的破折号，用于主句与从句



Demo_07 -- 放大/缩小pifont宏包字体
\documentclass{article}
\usepackage{graphicx}
\usepackage{pifont}
\let\oldding\ding
\renewcommand{\ding}[2][1]{\scalebox{#1}{\oldding{#2}}}
\begin{document}
\ding{172}\qquad\ding[1.2]{172}\\
\ding{192}\qquad\ding[1.2]{192}
\end{document}

内容讲解
1.\oldding保存原\ding内容

2.renewcommand{原命令}[参数数量][默认值]{定义内容}, #<n>代表引用使用命令时, 使用的第n个参数值
\scalebox包含在graphicx宏包中
