更新时间: 2020-07-10
1a.离线安装Texlive
# 下载texlive镜像
# 点击下载后，会自动切换到国内镜像源下载
# 下载地址：https://ctan.org/tex-archive/systems/texlive/Images  

# 校验完整性
steven@Downloads$ md5sum -c texlive2019-20190410.iso.md5

# 挂载iso文件
steven@Downloads$ sudo mount -o loop texlive2019-20190410.iso /mnt 

# 安装Texlive
steven@Downloads$ cd /mnt

steven@mnt$ sudo ./install-tl
安装选项列表，输入'i'/'I'进行安装


1b.在线安装Texlive
官网地址: http://tug.org/texlive/
选择'download' -- 点击'install-tl-unx.tar.gz'

解压install-tl-unx.tar.gz
tar -zxf install-tl-unx.tar.gz

安装
cd install-tl-*
./install-tl


2.配置PATH
steven@~$ vi .bashrc
export PATH=${PATH}:/usr/local/texlive/2020/bin/x86_64-linux

steven@~$ source .bashrc

# 测试安装是否成功
steven@~$ tex --version


3.TexLive字体查找配置
# 系统字体(windows会自动查找)
steven@~$ sudo vi /usr/local/texlive/2020/texmf.cnf
OSFONTDIR = /usr/share/fonts//;/usr/local/share/fonts//;~/.fonts//

# texlive附带字体
steven@~$ sudo cp /usr/local/texlive/2020/texmf-var/fonts/conf/texlive-fontconfig.conf /etc/fonts/conf.d/09-texlive.conf

# windows默认自带宋体/楷书/黑体/仿宋, ubuntu默认自带宋体/楷书
# 安装字体(黑体)
steven@~$ sudo apt install fonts-wqy-zenhei font-wqy-microhei

# 刷新fontconfig的字体缓存
steven@~$ sudo fc-cache -fsv

# 查看中文字体
steven@~$ sudo fc-list :lang=zh-cn


4.安装TeXworks编辑器
steven@~$ sudo add-apt-repository ppa:texworks/stable

steven@~$ sudo apt-get update

steven@~$ sudo apt-get install texworks

在launchpad中搜索texworks打开图形编辑界面
