octave -- 用于基础数学计算、线性代数计算和统计分析的软件


安装octave
sudo apt-get install octave

启动cli模式octave
octave [--no-gui]

启动gui模式octave
octave --gui



交互模式指令
exit/quit
	退出octave交互模式

help/doc <key>
	帮助文档
	
history
	历史指令列表
	
whos
	详细的变量信息列表
	
statement;
	完整语句, 抑制打印输出

statement
	完整语句
	


注释符
# %
	单行注释符
	
#{ #}
	块注释符
	
%{ %}
	块注释符
