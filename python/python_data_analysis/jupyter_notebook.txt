官网地址 - http://jupyter.org/
安装 - pip3 install jupyter
启动 - jupyter notebook

特征：
1.文件名后缀.ipynb
2.jupyter notebook的根目录源于终端当前目录
3.默认使用8888端口

配置文件默认路径: ~/.jupyter

jupyter notebook - 从默认浏览器启动jupyter

语法
jupyter notebook <command> [args]

command:
list
	列出当前运行的notebook服务

password
	为notebook设置密码

stop <port_number>
	关闭指定端口notebook，端口号默认为8888

args:
--allow-root
	允许notebook在root权限下允许

--debug
	调整log level为logging.DEBUG

--generate-config
	生成配置文件jupyter_notebook_config.py

--ip=<listen_ip>
	服务监听的IP

--no-mathjax
	关闭mathjax，mathjax为解释LaTeX的javascript库，占用带宽比较高

--no-browser
	启动服务后，不在浏览器中打开窗口

--port=<listen_port>
	服务监听的端口

-y
	默认yes

--pylab
	启用matplotlib，已弃用，替代方案为notebook环境中使用%pylab或%matplotlib


jupyter -

语法
jupyter [option]

参数释义
--config-dir
	当前配置文件路径

--paths
	可用配置文件搜寻路径




快捷键：
1.编辑模式：
shift+enter    执行单元
%connect_info    连接信息
%matplotlib notebook    启动matplotlib底层
ctrl+up    跳转到单元格起始
ctrl+down    跳转到单元格结束
2.指令模式：
Enter  进入编辑模式
Shift+Enter  执行单元并进入下一行(没有则新建)
Ctrl+Enter  执行单元
Alt+Enter  执行单元并新建下一行
J  下个cell
K  上个cell
Ctrl+S  检查并保存
Y  cell变成代码格式
M  cell变成markdown格式，使用html/LaTeX
1~6  cell变成h1~h6标题
R  清除cell格式
II  中断服务
00  重启服务
A  在上面插入cell
B  在下面插入cell
X  剪切cell(独立于system的剪切板)
C  复制cell
Shift+V  粘贴到上个cell(新增cell)
V  粘贴到下个cell(新增cell)
DD  删除选中cell
Z  撤销删除
H  快捷键菜单
