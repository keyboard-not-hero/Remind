IPython安装：
pip install ipython

原理
IPython采取client/server机制, 运算在server线程, client为面向用户线程


IPython特性:
1.特有的magic方法. 具体参考下列

2.可以使用操作系统指令, 语法: !<sys_cmd>. 如: !chmod



magic方法:
语法:
<magic_func> <python_statement>

magic_fun类型:
1.% - 行magic, 作用于当前行剩下内容
2.%% - 块magic, 作用于多行内容

magic_func列表:
%automagic
	使用magic时可以忽略'%'符号. 使用与行magic, 块magic仍然需要前置'%%'符号

%history
	历史输入指令

%pdef
	对象的定义头(包含方法名和形参)

%pdoc
	对象的注释文档(docstring)

%pfile
	该对象所在文件的全部内容

%psource
	对象的源内容

%run file <args>
	运行脚本, 并读取数据到当前namespace. 类似于python file <args>

** 具体magic列表参考%magic
** 单个magic详情参考%<magic_name>?



快捷键:
_/__/___
	上/上上/上上上一条输出

_i/_ii/_iii
	上/上上/上上上一条输入

ctrl+o
	在当前行下面新增一行



配置文件目录:
~/.ipython/profile_default


startup file - 启动ipython后马上执行的脚本, 文件后缀为.py或.ipy
目录: ~/.ipython/profile_default/startup/
