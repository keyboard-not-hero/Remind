安装系统
1.下载系统镜像
官网连接: https://www.raspberrypi.org/
进入官网, 选择"Software" -- 选择"Raspberry Pi OS" -- 选择"Raspberry Pi OS with desktop and recommended software" -- 点击"Download"

2.解压缩文件
unzip 2021-05-07-raspios-buster-armhf-full.zip

3.刻录镜像到sd卡
将sd卡插入读卡器(确保sd卡所在盘符)
sudo mkfs -t fat /dev/sdb
sudo dd if=~/Downloads/2021-05-07-raspios-buster-armhf-full.img of=/dev/sdb bs=4M conv=fsync

4.同步刷新
sync


升级系统
1.修改/etc/apt/sources.list和/etc/apt/sources.list.d/raspi.list
修改两个文件的主版本名称, 如: stretch --> buster

2.更新
sudo apt update
sudo apt -y dist-upgrade
