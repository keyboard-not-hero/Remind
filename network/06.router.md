开启/关闭接口<br>
`(config-if)# [no] shutdown`
<br>
<br>

给接口配置IP<br>
```
(config)# interface f0/1
(config-if)# ip address 192.168.1.2 255.255.255.0
```
<br>
<br>

查看接口的IP层简短信息<br>
`# show ip interface brief`
<br>
<br>

查看接口的IP层详细信息<br>
`# show ip interface`
<br>
<br>

查看IP层接口和路由启用情况<br>
`# show protocols`
