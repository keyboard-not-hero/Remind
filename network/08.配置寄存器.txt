启动过程:
1.开机自检POST(ROM)
2.使用启动程序(ROM)加载IOS, IOS查找顺序:
    (1)闪存(flash)
    (2)TFTP
    (3)ROM - 微IOS
3.加载启动配置文件, 启动配置文件查找顺序:
    (1)NVRAM
    (2)TFTP
    (3)查找不到, 进入设置模式


默认寄存器值: 0x2102
对应二进制值: 0010 0001 0000 0010

配置寄存器值:
(config)# config-register 0x2102

查看寄存器值:
Router> show version


不同bit寄存器意义:
第0~3 bit
    启动字段, 列表如下:
    0 - 进入ROM监视器模式(启动引导程序), 提示符rommon >, 并使用b手动启动
    ** 当无法从flash/tftp读取IOS image, 并且无法从rom中读取微型IOS时, 进入ROM监视器模式
    1 - 使用ROM中的微型IOS启动, 提示符Router(boot)>
    ** 当ROM中没有微型IOS时, 使用常规IOS启动
    2~F - 使用NVRAM中的boot命令启动
    ** boot命令列表:
        1)使用flash中指定的IOS启动(默认从查找到的第一个IOS启动)
        (config)# boot system flash <ios_name>

        2)使用tftp服务器中的IOS启动(当前不建议使用, tftp现常用于备份)
        (config)# boot system tftp <ios_name> <server_add>

        3)从rom加载微型IOS
        (config)# boot system rom
     查看boot序列:
     # show run | include boot system

第6 bit
    忽略NVRAM的内容

第7 bit
    启用OEM位
    
第8 bit
    忽略启动中断
    
第10 bit
    IP广播地址为全0地址

第5/11/12 bit
    控制台线路速度
