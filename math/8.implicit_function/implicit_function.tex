	1.隐函数求导\par
	\begin{align}
		2y+\sin(y)=\frac{x^2}{\pi}+1\\
		2\frac{dy}{dx}+\cos(y)\frac{dy}{dx}=\frac{2x}{\pi}\\
		\frac{dy}{dx}=\frac{2x}{\pi(2+\cos(y))}
	\end{align}
